
# 目标
开发一个叫做ccc的agent，类似codex和claude code 。 需要着重突出代码能力。

# 名称
ccc 的全称为 Cross Code Companion，中文名暂定为“跨模型代码助手”。

Cross 表示跨模型、跨工具、跨协议和跨工作流；Code 表示核心场景是代码理解、修改与验证；Companion 表示产品形态是长期陪伴开发者工作的本地代码智能体。

# 语言
前端用react， pnpm ， 后端用python


# session

第一版本只支持单用户的 多session，不需要多用户。
这里需要把用户的所有聊天都写入到磁盘中，目前是{pwd}/sessions/{sessions对应的uuid}，然后里面用jsonl来存储每一天的聊天记录，如果超过了10M，则分块

# 工具
目前只支持 skills/mcp/tools
tools 包括内置的工具以及扩展的工具

需要可以注册工具,最好是可扩展性

# memory
目前只需要超过一定阈值，然后compact，其他后面再说


# 通信方式
目前还是http即可

# 前端
第一版使用基于 React 的 TUI 风格 Web 前端，视觉和交互上类似 claude code 和 codex 的终端体验，但运行形态是 Web UI，不是真正运行在终端里的 TUI。

后续如需要真正的终端入口，可增加 Ink/CLI 前端，通过同一套 HTTP API 或后端入口接入 Python Agent 后端。
